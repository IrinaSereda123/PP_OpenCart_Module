Часть 1. Теоретическая база

Архитектура MVC‑L

OpenCart построен на шаблоне MVC‑L (Model‑View‑Controller‑Language). Функциональность разделена на четыре слоя:

  • Model – работа с базой данных и бизнес‑логика. Без HTML и прямых запросов.
  • View – шаблоны (Twig). Только отображение данных.
  • Controller – обработка запросов, загрузка моделей, языковых файлов, передача данных во view.
  • Language – текстовые константы для локализации.

Структура каталогов модуля (на примере my_module)

my_module/
├── admin/
│   ├── controller/extension/module/my_module.php
│   ├── model/extension/module/my_module.php
│   ├── language/ru-ru/extension/module/my_module.php
│   └── view/template/extension/module/my_module.twig
└── catalog/
    ├── controller/extension/module/my_module.php
    ├── model/extension/module/my_module.php
    ├── language/ru-ru/extension/module/my_module.php
    └── view/theme/default/template/extension/module/my_module.twig

Типы расширений

  • Платежные шлюзы
  • Методы доставки
  • Итоги заказа (скидки, наценки)
  • Каналы товаров (импорт/экспорт)
  • Отчёты
  • Модификации OCMOD и система событий (Events)
OCMOD vs События
OCMOD (XML‑модификации) выполняет поиск/замену в файлах – менее надёжно.
Система событий (events) – современный подход, не изменяет файлы ядра, подключается к точкам входа.

Часть 2. Пошаговое руководство. Модуль «Hello World»

Создадим модуль, который выводит настраиваемый текст в левой колонке.

Подготовка

  • Локальный сервер (OpenServer, XAMPP).
  • Включение отображения ошибок: админ‑панель → Система → Настройки → вкладка «Сервер» → «Показывать ошибки» = Да. (После отладки выключить).
  • Имя модуля: hello_world.

admin/controller/extension/module/hello_world.php
<?php
class ControllerExtensionModuleHelloWorld extends Controller {
    private $error = array();
    public function index() {
        $this->load->language('extension/module/hello_world');
        $this->document->setTitle($this->language->get('heading_title'));
        $this->load->model('setting/module');
        if (($this->request->server['REQUEST_METHOD'] == 'POST') && $this->validate()) {
            if (!isset($this->request->get['module_id']))
                $this->model_setting_module->addModule('hello_world', $this->request->post);
            else
                $this->model_setting_module->editModule($this->request->get['module_id'], $this->request->post);
            $this->session->data['success'] = $this->language->get('text_success');
            $this->response->redirect($this->url->link('marketplace/extension', 'user_token=' . $this->session->data['user_token'] . '&type=module', true));
        }
        $data['heading_title'] = $this->language->get('heading_title');
        $data['text_edit'] = $this->language->get('text_edit');
        $data['entry_name'] = $this->language->get('entry_name');
        $data['entry_title'] = $this->language->get('entry_title');
        $data['entry_message'] = $this->language->get('entry_message');
        $data['entry_status'] = $this->language->get('entry_status');
        $data['button_save'] = $this->language->get('button_save');
        $data['button_cancel'] = $this->language->get('button_cancel');
        $data['action'] = $this->url->link('extension/module/hello_world', 'user_token=' . $this->session->data['user_token'], true);
        $data['cancel'] = $this->url->link('marketplace/extension', 'user_token=' . $this->session->data['user_token'] . '&type=module', true);
        if (isset($this->request->get['module_id']) && ($this->request->server['REQUEST_METHOD'] != 'POST')) {
            $module_info = $this->model_setting_module->getModule($this->request->get['module_id']);
        }
        $data['name'] = $module_info['name'];
        $data['title'] = $module_info['title'];
        $data['message'] = $module_info['message'];
        $data['status'] = $module_info['status'];
        $data['header'] = $this->load->controller('common/header');
        $data['column_left'] = $this->load->controller('common/column_left');
        $data['footer'] = $this->load->controller('common/footer');
        $this->response->setOutput($this->load->view('extension/module/hello_world', $data));
    }
    protected function validate() {
        if (!$this->user->hasPermission('modify', 'extension/module/hello_world'))
            $this->error['warning'] = $this->language->get('error_permission');
        return !$this->error;
    }
}

Фронтальная часть

catalog/controller/extension/module/hello_world.php
<?php
class ControllerExtensionModuleHelloWorld extends Controller {
    public function index($setting) {
        $data['message'] = isset($setting['message']) && !empty($setting['message']) 
            ? html_entity_decode($setting['message'], ENT_QUOTES, 'UTF-8')
            : 'Текст не задан в настройках модуля.';
        $data['title'] = $setting['title'] ?? '';
        return $this->load->view('extension/module/hello_world', $data);
    }
}

Установка и активация

  1. Упаковать папку hello_world в ZIP‑архив.
  2. Админ‑панель → Расширения → Установщик расширений → загрузить архив.
  3. Расширения → Расширения → тип «Модули» → найти модуль → зелёная кнопка «Установить».
  4. Редактировать модуль, задать название, заголовок, текст, включить статус.
  5. Дизайн → Макеты → выбрать макет (Default) → добавить модуль, указать позицию (Column Left) и сортировку.

Часть 3. Индивидуальные задания (15 работ)

Каждое задание развивает базовый модуль «Hello World».

1 Многоязычность

Добавить в админ‑часть возможность ввода текста сообщения отдельно для каждого языка (русский/английский) с помощью мультиязычных полей.

2 Изображение

Добавить в настройки модуля поле для загрузки изображения и выводить его на витрине вместе с текстом.

3 База данных

Создать таблицу oc_hello_world_quotes (id, text). Разработать модель для работы с ней. Выводить случайную запись из таблицы.

4 Выбор позиции

Добавить в настройки выпадающий список позиций (левая колонка, правая колонка, контент) и динамически размещать модуль согласно выбору.

5 Стилизация

Подключить отдельный CSS‑файл модуля, изменить фон, шрифт, отступы блока.

6 AJAX‑обновление

Добавить кнопку «Обновить цитату», которая без перезагрузки страницы загружает новую случайную запись из БД (на основе задания 3).

7 Выбор цвета

В админ‑настройках добавить поле выбора цвета фона (color picker). Цвет применяется к блоку модуля на витрине.

8 Ссылка в модуле

Добавить поле «URL ссылки» и «Текст ссылки». Если заполнены, выводить кликабельный элемент под сообщением.

9 Переключатель заголовка

Добавить опцию «Показывать заголовок» (да/нет). При отключении заголовок не выводится.

10 Иконка Font Awesome

Позволить выбирать иконку из списка (например, fa-smile, fa-star) и отображать её перед заголовком.

11 Кэширование

Реализовать кэширование вывода модуля на 5 минут. Использовать метод $this->cache->get()/set().

12 CSS‑классы

Добавить поле «Дополнительные CSS‑классы» в настройках модуля и применить их к контейнеру модуля.

13 Ограничение длины текста

Добавить настройку «Максимальная длина сообщения». Обрезать текст, если он превышает лимит, добавлять многоточие.

14 Даты обновления

При сохранении настроек модуля фиксировать дату последнего изменения и выводить её в блоке модуля («Последнее обновление: …»).

15 Упаковка и документация

Создать install.json, подготовить модуль к публикации, написать краткую документацию (установка, настройка).

Код должен соответствовать стандартам кодирования OpenCart. Результат предоставляется в виде отчета, архива с файлами и инструкцией по установке.
